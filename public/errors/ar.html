Errors

This document identifies some of the error codes and messages that OMONO's API return.

This page lists errors by their HTTP status codes as defined in RFC https://tools.ietf.org/html/rfc7231.

Error is follewed standards defined in RFC https://tools.ietf.org/html/rfc7807

  1. NOT_FOUND (404)
  2. INTERNAL_SERVER_ERROR (500)
